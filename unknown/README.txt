If a pokemon is  here follow these steps
1.Go to https://bulbapedia.bulbagarden.net
2.Download the image for it directly from the sidebar and name the file pokemonname.jpg
3.Delete the pokedex.dat and run the program. It should generate a new one with the pokemon
4.(Optional) Let me know via Github by creating an issue or by messaging me so I can add it to the default list
